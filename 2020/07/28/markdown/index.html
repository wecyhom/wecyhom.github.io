Markdown

发表于 2020-07-28

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。

你将了解到：

  • Markdown与传统格式化方法的不同
  • 如何使用Markdown格式化文本

    什么是Markdown？

    Markdown是一种在网络上设置文字样式的方法。你可以控制文档的显示、将文本格式化位粗体或斜体、添加图像或创建列表。通常，Markdown还会带有非字母字符文本，例如#或*

标题

有时候使用不同等级的标题来结构化你的文档非常有用。

在标题前添加`#`来确定等级，连续多个`##`表示较小等级的标题

### 这是三级标题

#### 这是四级标题
